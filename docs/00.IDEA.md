1. MonoChat
	- 開発
		- 言語：Ruby
		- フレームワーク：Ruby on Rails
	- デザイン
		- 全てモノクロ
		- 匿名掲示板
		- フォント：[Google Fonts | DotGothic16](https://fonts.google.com/specimen/DotGothic16)
		- アプリ内使用アイコン：[Tabler Icons](https://tabler.io/icons)
	- 匿名認証でログインする
	- RailsのDBをそのまま流用して実装を簡略化
	- 用語、言い回し
		- ログイン --> Login
		- ホーム画面 --> Home
		- 掲示板 --> Space
	- ログイン (Login)
		- ユーザーはパスワードのみを使用してログイン
			- ユーザー名を決定することはできない
			- ユーザー名はバックエンドで自動的に生成されDBに格納
		- 初回登録
			- フロントエンド-->バックエンドにパスワードのみを送信
			- バックエンド-->パスワードを受け取り、(UUID, ユーザー名)を生成、パスワードをハッシュ化して全てをDBに保存
		- ログイン
			- フロントエンド-->パスワードのみ入力してログイン。UUIDはブラウザに保存されているものを使用する。
			- バックエンド--> フロントから来たUUID / Passwordで認証を行い、認可が下りるとJWTをフロントに返しログイン完了
	- メッセージ投稿 (Message)
		- 5秒のレート制限
	- ホーム画面 (Home)
		- ヘッダー
			- アプリタイトル (左)
			- ユーザー名 (右)
		- ボディ
			- 掲示板一覧
			- 掲示板作成ボタン (New Space)
				- ポップアップウィンドウ
					- 作成承認ボタン
					- --> 掲示板名を決めることはできない
		- フッダー
			- Author, Github Repo, Ruby on Rails (右寄せ)
	- 掲示板画面 (Space)
		- ヘッダー・フッダー
			- == Home
		- ボディ
			- メッセージ
				- 左寄せ
				- 文字だけのシンプルなテキストボックス
				- 内側白、枠黒、角
				- メッセージ上部にユーザー名
			- 入力ボックス
				- 画面下中央
				- 内側白、枠グレー、角
				- 右下に送信ボタン
					- 送信処理中はグレーアウト
	- データベース
		- ユーザー (users)
			- ID (integer) --> id
			- UUID (uuid4) --> user_uuid
			- ユーザー名 (string) --> user_name
			- パスワード (string -- bcrypt) --> password_digest
			- 作成日時 (datetime) --> created_at
			- 更新日時 (datetime) --> updated_at
		- 掲示板 (spaces)
			- ID (integer) --> id
			- UUID (uuid4) --> space_uuid
			- 作成日時 (datetime) --> created_at
			- 更新日時 (datetime) --> updated_at
		- メッセージ  (messages)
			- メッセージUUID (UUID4) --> message_uuid
			- 送信ユーザーUUID (UUID4) --> sender_uuid
			- 掲示板UUID  --> space_uuid
			- 内容 (Content) --> content
			- 作成日時 (datetime) --> created_at
			- 更新日時 (datetime) --> updated_at
