## 1. 概要
MonoChatは、全てがモノクロで構成された匿名掲示板アプリケーションです。
デザインには「DotGothic16」フォントと「Tabler Icons」を使用し、独特な世界観を表現しています。
ユーザー登録は不要で、パスワードのみを使用した匿名認証により、手軽に利用を開始できます。

- Ruby on Rails

## 3. 使い方

### ログイン (Login)
- **匿名認証**: ユーザー名の設定は不要です。バックエンドで自動的にユニークなIDが生成されます。
- **ログイン手順**: 初回はパスワードを入力して登録します。2回目以降も同じパスワードを入力するだけでログイン可能です。

### ホーム画面 (Home)
- **掲示板一覧**: 作成された掲示板（Space）の一覧が表示されます。
- **新規作成**: 「New Space」ボタンから新しい掲示板を作成できます。掲示板名も自動生成されます。

### 掲示板 (Space)
- **メッセージ送信**: シンプルなテキストメッセージを送信できます。
- **レート制限**: 連続投稿を防ぐため、5秒間の送信制限があります。
